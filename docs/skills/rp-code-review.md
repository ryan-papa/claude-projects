# rp-code-review

코드 리뷰 (8항목 평가).

## 트리거

- QA 통과 후 자동
- `/rp-code-review` 명령

## 유형별 분기

| 유형 | 리뷰 방식 |
|------|----------|
| 코드 프로젝트 | 8항목 전체 평가 |
| 콘텐츠 프로젝트 | 빌드 검증만 (QA 검수에서 품질 확인 완료) |

## 평가 항목 (코드 프로젝트, 각 10점)

정확성 / 보안 / 성능 / 설계 / 가독성 / 테스트 / 동시성 / 운영 가시성

## PR 유형별 포커스

| PR 유형 | 집중 영역 |
|---------|-----------|
| 버그 픽스 | 정확성 + 테스트 |
| 새 기능 | 설계 + 보안 + 성능 |
| 리팩토링 | 가독성 + 테스트 |

## 판정

- 평균 >= 8.0 → 통과
- < 8.0 → Dev Agent 재투입 (최대 3회)
- 3회 후 < 7.0 → 사용자에게 추가 사이클 여부 확인
- 3회 후 7.0~8.0 → 통과 처리

## 이슈 처리

| 심각도 | 처리 |
|--------|------|
| High/Mid | Dev Agent가 자동 수정 |
| Low | 테이블로 제시, 사용자 선택 |

→ 리뷰 기준 상세: [`../harness-code-review.md`](../harness-code-review.md)
